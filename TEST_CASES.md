# 电瓶车租赁管理系统 - 测试用例

## 测试环境
- 微信开发者工具
- 微信云开发环境

---

## 1. 车辆管理

### 1.1 添加车辆
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 正常添加 | 1. 进入"我的车库" 2. 点击"+" 3. 填写车辆信息（名称、品牌、车架号、车牌、颜色、电池） 4. 上传封面图 5. 点击保存 | 车辆添加成功，跳转车库页面，新车辆显示在列表中 |
| 必填字段缺失 | 填写部分必填字段后点击保存 | 提示"请输入xxx"，不跳转 |
| 无封面图 | 不上传封面图，直接保存 | 可以保存，使用默认背景 |
| 重复车牌号 | 输入已存在的车牌号保存 | 可以保存（系统不校验唯一性）|

### 1.2 车辆列表
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 首次加载 | 进入车库页面 | 显示loading，数据加载完成后展示 |
| 空列表 | 无任何车辆时进入 | 显示空状态图片，不显示暂无车辆闪烁 |
| 切换标签 | 点击"空闲"/"租聘中"/"待结算" | 列表按状态筛选显示 |
| 筛选结果为空 | 某状态下无车辆 | 显示空状态图片 |

### 1.3 编辑车辆
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 编辑空闲车辆 | 1. 点击车辆卡片 2. 点击编辑 3. 修改信息 4. 保存 | 信息更新成功 |
| 编辑租聘中车辆 | 编辑非空闲状态的车辆 | 可以编辑（系统未限制） |
| 编辑他人车辆 | （理论上无法操作，权限校验在详情页） | 无权操作提示 |

### 1.4 删除车辆
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 删除空闲车辆 | 1. 点击删除 2. 确认 | 车辆删除，列表刷新 |
| 删除租聘中车辆 | 删除有进行中订单的车辆 | 可以删除（订单状态不受影响）|

---

## 2. 租聘流程

### 2.1 租车
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 正常租车流程 | 1. 首页点击"租聘" 2. 选择车辆 3. 填写租车人信息 4. 上传身份证正反面 5. 选择租车时长 6. 点击确认 | 订单创建成功，车辆状态变为"租聘中"，跳转车库页面 |
| 必填字段缺失 | 未填写租车人姓名/手机号 | 提示"请输入xxx"，无法提交 |
| 手机号格式错误 | 输入非1开头的11位数字 | 提示"手机号码格式不正确" |
| 未上传身份证 | 不上传身份证照片 | 提示"请上传身份证" |
| 选择时长 | 点击快捷时长选项（如7天、30天） | 到期时间自动计算 |
| 自定义时长 | 选择自定义，输入天数 | 到期时间自动计算 |
| 订阅消息 | 点击确认后弹出订阅消息授权 | 无论是否订阅都继续保存流程 |

### 2.2 续租
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 正常续租 | 1. 首页点击"续租" 2. 选择租聘中的车辆 3. 选择续租时长 4. 确认 | 续租成功，到期时间更新，跳转车库页面 |
| 续租时长为0 | 选择0天或输入0 | 提示"请选择续租时长" |

### 2.3 结算
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 正常结算 | 1. 首页点击"结算" 2. 选择待结算车辆 3. 上传结算照片 4. 填写备注 5. 确认 | 结算成功，车辆状态变为"空闲"，跳转车库页面 |
| 无租聘记录 | 对没有进行中订单的车辆进行结算 | 提示"无有效租聘记录" |
| 无结算照片 | 不上传结算照片 | 可以结算（照片非必填）|

## 3. 订单管理

### 3.1 当前订单
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 查看当前订单 | 1. 点击底部"订单"tab | 显示进行中的租聘订单列表 |

### 3.2 历史订单
| 用例 | 步骤 | 预期结果 |

| 查看历史订单 | 1. 进入历史订单页面 | 显示所有历史订单（按时间倒序） |

|------|------|---------|

## 6. 云函数

### 6.1 获取OpenId
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 正常获取 | 小程序启动 | 自动调用getOpenId云函数，获取用户openId并缓存 |

### 6.2 过期订单检测
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 有过期订单 | 云函数被触发且有待过期订单 | 发送订阅消息通知用户，订单状态变为"待结算" |
| 无过期订单 | 云函数被触发且无过期订单 | 无操作，返回success |
| 订单无renterOpenId | 订单缺少openId字段 | 跳过该订单，不发送消息 |

## 8. 异常场景

### 8.2 数据异常
| 用例 | 步骤 | 预期结果 |
|------|------|---------|
| 车辆详情异常 | car集合数据被物理删除但订单关联 | 页面显示异常数据提示 |
