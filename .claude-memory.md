## 当前进度

### 已完成改造

1. **静默登录** - openId 自动获取，存入 globalData.openId
2. **用户体系** - user 表存储用户信息（名字、头像）
3. **权限隔离** - create_by 字段隔离数据
4. **操作日志** - 每次增删改记录到 operation_log
5. **车辆状态流转**：
   - 0（空闲）→ 商家手动"租出"按钮
   - 1（租聘中）→ checkExpiredRentals 云函数检测过期自动变更为 2（待结算）
   - 2（待结算）→ 结算后自动变更为 0（空闲）

### 首页功能按钮
- 🔑 租聘 (carStatus=0) → 车库只显示空闲车辆
- 🔄 续租 (carStatus=1) → 车库只显示租聘中车辆
- 🧾 结算 (carStatus=2) → 车库只显示待结算车辆

### 核心页面流程
```
首页 → 租聘/续租/结算 → 车库(选择模式) → 对应操作页面
```
